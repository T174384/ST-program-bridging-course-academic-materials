# ST-program-bridging-course-academic-materials
恭喜来到这个仓库的学弟学妹们，你们即将在NUS预科开启一段崭新的学习之旅，按照2025年第二届ST的学习过程来看，学期上会学习英语，Basic C,数学，化学，学期下会学习Advance C,物理，英语。
此仓库会包含预科课程的几乎所有资料，涵盖计算机，物理，化学，数学的PPT，tutorial和对应答案；课堂笔记，模拟测试题和对应答案，Lab, project等等编程代码
从我个人角度来看预科的课程思维难度不大，考试难度重在记背，几乎无思维难度，但尤其需要注意的是学期上化学的Check point1可能会很难（我们这一届真的很难），是Zhangsheng老师出的题，至于课堂，如果你的学习能力在线，并且觉得接受不了老师的授课方式，自学也完全不会有问题，尤其是物理，C语言。鄙人就是不大喜欢Tay教授的节奏和逻辑选择自学，也取得了相当不错的成绩，这里关于自学有几点建议，一定要认真把PPT的内容自己完整看一遍，不懂的查询AI，Tutorial认真做加订正答案，重复的习题查询答案可以参考这个仓库，每个Lab就算不会，也要在参考完标准答案后亲自写一遍。如果你觉得把所有时间浪费在预科学习上不值得，也可以选择平时水水学临时抱佛脚的模式，但一定也有平时的最低学习保障，PPT自学和Tutorial ,Lab这几样，然后考前突击复习（考前复习可以参考这个仓库里的对应笔记）。
最后很诚挚的邀请学弟学妹们，十分欢迎那些有意愿成为这个仓库管理员的人，可以不断更新完善这些资料，帮助到后面来NUS求学的ST计划的预科学生。
